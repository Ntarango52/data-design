Conceptual Model

User

  • UserId (Primary Key)
  • Gender
  • Name
  • Email
  • Debit/Credit Card

Profile

  • ProfileId (Primary Key)
  • AccountId (Foreign Key)
  • ProfilePreferences
  • ProfileDate

Show

  • ShowId (Foreign Key)
  • ShowContent
  • ShowLanguage

Relationship

  • One user can have one profile (1 to 1)
  • One profile can have many shows and movies (1 to many)
