Conceptual Model

User

  • userId (Primary Key)
  • userActivationToken
  • userName
  • userEmail

Profile

  • profileId (Primary Key)
  • profileUserId (Foreign Key)
  • profileDate

Media

  • mediaUserId (Foreign Key)
  • mediaProfileId (Foreign Key)
  • mediaContent

Relationship

  • One user can have many profiles (1 to many)
  • One profile can have many shows and movies (1 to many)
