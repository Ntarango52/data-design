Conceptual Model

User

  • userId (Primary Key)
  • userActivationToken
  • userName
  • userEmail

Profile

  • profileId (Primary Key)
  • profileUserId (Foreign Key)
  • profilePreferences
  • profileDate

Show

  • showUserId (Foreign Key)
  • showProfileId (Foreign Key)
  • showContent
  • showLanguage

Relationship

  • One user can have one profile (1 to 1)
  • One profile can have many shows and movies (1 to many)
